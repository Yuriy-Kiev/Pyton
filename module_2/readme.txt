1. Задача проекта - освоить и закрепить навыки проведения разведовательного анализа EDA, 
   подготовить датасет для построения модели взаимосвязи оценки по математике с различными показателями,
   описывающими условия учебы школьников.

2. Изначальний датасет содержит информацию о 29 показателях, описывающих условия учебы школьников.

3. Основные этапы выполнения проекта:

  3.1. Анализ данных каждого столбца, подсчет количества пропусков, устранение ошибок ввода данных.

  3.2. Проведение корреляционного анализа данных, устранение столбцов, содержащих зависимые данные.

  3.3. Устранение выбросов и анализ влияния этой процедуры на показатели корреляции.

  3.4. Определение и устранение статистически незначимых номинативных показателей.

  3.5. Формирование выводов и постановка гипотез относительно данных в датасете.

4. Вопросы для саморефлексии:

  4.1. Какова была ваша роль в команде?
  Ответ: командной работой можно считать только общение в slack относительно заданий проекта.

  4.2. Какой частью своей работы вы остались особенно довольны?
  Ответ: всей работой доволен, освоим множество новых методов, не описанных в учебном курсе.

  4.3. Что не получилось сделать так, как хотелось? Над чем ещё стоит поработать?
  Ответ: не успел пройти не обязательную программу по статистике. Буду изучать на каникулах.

  4.4. Что интересного и полезного вы узнали в этом модуле?
  Ответ: методы подготовки данных для модели, интересные способы визуализации данных, использование
	графиков для анализа выбросов и зависимостей.

  4.5. Что является вашим главным результатом при прохождении этого проекта?
  Ответ: освоение и закрепление навыков EDA.

  4.6. Какие навыки вы уже можете применить в текущей деятельности?
  Ответ: выявлять статистически значимые номинативные признаки (влияние типов брака на превышение
	тех. отхода).

  4.7. Планируете ли вы дополнительно изучать материалы по теме проекта?
  Ответ: да, даже приобрел учебники по статистике.
